# Offline Pokédex

An offline Pokédex application built with **Python** and **PyQt**.
The app displays Pokémon information including stats, types, weaknesses,
abilities, and evolutions using a local dataset.

## Features
- Offline Pokémon database
- Detailed stats (HP, Attack, Defense, Sp. Atk, Sp. Def, Speed)
- Type & weakness visualization
- Evolution chain support
- Fast UI with multithreading
- Clean PyQt-based interface
